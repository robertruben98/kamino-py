# kamino-py

Python client for the Kamino Finance API (Solana lending, vaults, liquidity strategies). Sync + async, Pydantic v2.

## Install

```bash
pip install kamino-py
```

The distribution is `kamino-py`; the import package is `kamino`.

## Quick start

```python
import kamino
```

See the [API reference](api.md) for the full surface.

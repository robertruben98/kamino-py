    API reference

    kamino

    kamino-py: a typed Python client for the Kamino Finance public API.

    Provides synchronous (:class:KaminoClient) and asynchronous (:class:AsyncKaminoClient) clients for read-only access to Kamino Finance data on Solana: lending markets, reserves, liquidity vaults, oracle prices, and staking yields.

    AsyncKaminoClient

    Asynchronous client for the Kamino Finance public API.

    Mirrors :class:KaminoClient but exposes coroutine methods and wraps an :class:httpx.AsyncClient.

    Parameters:

    Name Type Description Default
    base_url str

    API base URL. Defaults to https://api.kamino.finance. Any trailing slash is stripped.

    		 DEFAULT_BASE_URL
    timeout float

    Request timeout in seconds. Defaults to 30.

    		 DEFAULT_TIMEOUT
    http_client Optional[AsyncClient]

    An optional pre-configured :class:httpx.AsyncClient. When provided, the caller is responsible for closing it.

    		 None

    Use as an async context manager::

    async with AsyncKaminoClient() as client:
    markets = await client.get_markets()

    aclose() async

    Close the underlying async HTTP client if owned by this instance.

    get_markets(*, program_id=None) async

    Get the configuration of all Kamino lending markets.

    Calls GET /v2/kamino-market. See :meth:KaminoClient.get_markets.

    get_market(pubkey, *, program_id=None) async

    Get a single lending market config.

    Calls GET /v2/kamino-market/{pubkey}. See :meth:KaminoClient.get_market.

    get_reserve_metrics(market_pubkey) async

    Get current metrics for all reserves in a lending market.

    Calls GET /kamino-market/{pubkey}/reserves/metrics. See :meth:KaminoClient.get_reserve_metrics.

    get_vaults() async

    Get all Kamino liquidity vaults (KVaults).

    Calls GET /kvaults/vaults. See :meth:KaminoClient.get_vaults.

    get_vault(pubkey) async

    Get a single Kamino liquidity vault by address.

    Calls GET /kvaults/vaults/{pubkey}. See :meth:KaminoClient.get_vault.

    get_vault_metrics(pubkey) async

    Get current metrics for a vault.

    Calls GET /kvaults/vaults/{pubkey}/metrics. See :meth:KaminoClient.get_vault_metrics.

    get_oracle_prices(*, markets='main') async

    Get Kamino oracle prices.

    Calls GET /oracles/prices. The markets argument selects the market set: "main" (default) or "all". See :meth:KaminoClient.get_oracle_prices.

    get_staking_yields() async

    Get staking yields for supported tokens.

    Calls GET /v2/staking-yields. See :meth:KaminoClient.get_staking_yields.

    KaminoClient

    Synchronous client for the Kamino Finance public API.

    Parameters:

    Name Type Description Default
    base_url str

    API base URL. Defaults to https://api.kamino.finance. Any trailing slash is stripped.

    		 DEFAULT_BASE_URL
    timeout float

    Request timeout in seconds. Defaults to 30.

    		 DEFAULT_TIMEOUT
    http_client Optional[Client]

    An optional pre-configured :class:httpx.Client. When provided, the caller is responsible for closing it; otherwise the client manages its own and closes it on :meth:close.

    		 None

    The client is safe to use as a context manager::

    with KaminoClient() as client:
    markets = client.get_markets()

    close()

    Close the underlying HTTP client if it is owned by this instance.

    get_oracle_prices(*, markets='main')

    Get Kamino oracle prices.

    Calls GET /oracles/prices.

    Parameters:

    Name Type Description Default
    markets Literal['main', 'all']

    Which market set to query. "main" (the default) returns oracle prices for assets in the main market; "all" returns prices for assets across all public KLend markets. The API only accepts these two values (passing anything else returns HTTP 400).

    		 'main'

    Returns:

    Type Description
    list[OraclePrice]

    A list of :class:~kamino.models.OraclePrice.

    KaminoModel

    Bases: BaseModel

    Base model shared by all Kamino response models.

    Configures pydantic to accept unknown fields (forward compatibility), populate fields by either alias or attribute name, and keep extra fields accessible via :attr:~pydantic.BaseModel.model_extra.

    KvaultMetrics

    Bases: KaminoModel

    Current metrics for a Kamino liquidity vault (KVault).

    Returned by GET /kvaults/vaults/{pubkey}/metrics.

    All APY and amount fields are high-precision decimal strings. Only the most commonly used fields are declared explicitly; any additional fields returned by the API remain available via :attr:~pydantic.BaseModel.model_extra.

    Attributes:

    Name Type Description
    apy Optional[str]

    Annualized percentage yield (decimal string).
    apy_24h Optional[str]

    Trailing 24-hour APY (decimal string).
    apy_7d Optional[str]

    Trailing 7-day APY (decimal string).
    apy_30d Optional[str]

    Trailing 30-day APY (decimal string).
    apy_90d Optional[str]

    Trailing 90-day APY (decimal string).
    apy_365d Optional[str]

    Trailing 365-day APY (decimal string).
    token_price Optional[str]

    Underlying token price in USD (decimal string).
    share_price Optional[str]

    Vault share price (decimal string).
    tokens_available Optional[str]

    Idle tokens available in the vault (decimal string).
    tokens_invested Optional[str]

    Tokens currently deployed in strategies (decimal string).
    number_of_holders Optional[int]

    Number of distinct vault share holders.

    MarketConfig

    Bases: KaminoModel

    Configuration for a single Kamino lending market.

    Returned by GET /v2/kamino-market (as a list) and GET /v2/kamino-market/{pubkey} (single).

    Attributes:

    Name Type Description
    name Optional[str]

    Human-readable market name (e.g. "Main Market").
    lending_market Optional[str]

    Base58 public key of the lending market account.
    is_primary Optional[bool]

    Whether this is the protocol's primary market.
    description Optional[str]

    Free-form description of the market.
    lookup_table Optional[str]

    Base58 address of the market's address lookup table.
    is_curated Optional[bool]

    Whether the market is a curated (third-party) market.

    OraclePrice

    Bases: KaminoModel

    A single oracle price entry.

    Returned by GET /oracles/prices (as a list when no markets filter is applied).

    Attributes:

    Name Type Description
    mint Optional[str]

    Base58 mint address of the priced token.
    name Optional[str]

    Token symbol (e.g. "EURC").
    price Optional[str]

    Token price as a high-precision decimal string.
    max_age_in_seconds Optional[str]

    Maximum acceptable staleness of the price, in seconds (string).
    timestamp Optional[str]

    Unix timestamp of the price observation, as a string.

    ReserveMetrics

    Bases: KaminoModel

    Current metrics for a single reserve within a lending market.

    Returned by GET /kamino-market/{pubkey}/reserves/metrics (as a list).

    Numeric values are high-precision decimal strings as provided by the API.

    Attributes:

    Name Type Description
    reserve Optional[str]

    Base58 public key of the reserve account.
    liquidity_token Optional[str]

    Symbol of the reserve's liquidity token (e.g. "USDC").
    liquidity_token_mint Optional[str]

    Base58 mint address of the liquidity token.
    max_ltv Optional[str]

    Maximum loan-to-value ratio for the reserve, as a decimal string.
    borrow_apy Optional[str]

    Current borrow APY, as a decimal string (e.g. "0.04075").
    supply_apy Optional[str]

    Current supply APY, as a decimal string.
    total_supply Optional[str]

    Total supplied amount in token units, as a decimal string.
    total_borrow Optional[str]

    Total borrowed amount in token units, as a decimal string.
    total_supply_usd Optional[str]

    Total supplied value in USD, as a decimal string.
    total_borrow_usd Optional[str]

    Total borrowed value in USD, as a decimal string.
